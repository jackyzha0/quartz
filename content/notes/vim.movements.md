|                                    |                                              |
| ---------------------------------- | -------------------------------------------- |
| h j k l                            |    ⃪ ↓ ↑ →                                      |
| 0                                  | to the `start` of the line                   |
| $ (Shit + 4)                       | to the `end` of the line                     |
| e                                  | to `end` of word                             |
| w                                  |`forward` one **word**                       |
| W                                  |`forward` one **Word**                       |
| b                                  |`backward` one **word**                      |
| B                                  |`backword` one **Word**                      |
| *<`number`>* wW / e / bB / h j k l | Move *<`number`>* times *<`text objects`>* |
| *<`number`>* G                         | to **specific line**                             |
| %                                  | to **matching parenthesis {} () []**             |
| zz                                 | re-center                                             |
